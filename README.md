# 키워드 커리어 진단

RIASEC 기반의 키워드 커리어 진단 웹 애플리케이션입니다.

## 기능

- 120개의 키워드를 통해 RIASEC 성향 분석
- 핵심 키워드 3개 선택
- 직업 가치관 선택
- 종합 진단 결과 대시보드 제공

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

### 빌드

```bash
npm run build
```

### 미리보기

```bash
npm run preview
```

## 기술 스택

- React 18
- Vite
- Tailwind CSS
- Lucide React (아이콘)
