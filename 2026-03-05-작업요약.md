## 2. 전공 / 이전 직장 직무 입력 필드 추가

AI 프롬프트 품질 향상을 위해 **전공**과 **이전 직장 직무** 정보를 기입할 수 있는 입력 필드를 추가.  
두 필드 **모두 선택 기입 사항**이며, 비어 있으면 프롬프트에서 해당 내용이 자동 생략됨.

### 변경 파일

#### `TCITest.jsx`
- `userInfo` state 추가 (`major`, `pastJob`)
- TCI 점수 입력 폼 **상단**에 "기본 배경 정보" 섹션 카드 삽입
- `HollandTest`로 넘기는 `tciContext`에 `major`, `pastJob` 포함

#### `BirkmanTest.jsx`
- `userInfo` state 추가 (`major`, `pastJob`)
- 인트로 → 컬러 선택 사이에 **`background` 스텝** 신규 추가
  - 전공 / 이전 직장 직무 입력 후 다음 단계로 이동
- `handleBack` 에 `background` 스텝 연결
- `HollandTest`로 넘기는 `birkmanContext`에 `major`, `pastJob` 포함

#### `HollandTest.jsx`
- `tciContext` 또는 `birkmanContext`에서 `major`, `pastJob` 추출
- 입력된 값만 `[배경 정보]` 블록으로 AI 프롬프트 상단에 삽입
  ```
  [배경 정보]
  전공: 경영학
  이전 직장 직무: 영업관리
  ```
- 모든 프롬프트(직업 추천, 환경, 상사, 동료)에 공통 적용

### UI 문구
- 라벨: `전공 (선택 기입 사항)`, `이전 직장 직무 (선택 기입 사항)`
- 안내 문구: "모두 선택 사항이니 해당 없으면 비워두고 다음으로 넘어가주세요."
